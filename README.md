## Installation
```terminal
sh -c "`curl -fsSL https://raw.githubusercontent.com/chankaward/dotfiles/master/install.sh`"
```
### Install Vim Plugins
``` terminal
cd ~/.dotfiles
mvim vimrc
```
in ```vimrc``` run
```terminal
PluginInstall
```

After installing, open a new terminal window to see the effects.
